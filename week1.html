brief introduction to software development

   COLOSSUS   

  • world’s first programmable electronic, digital computer

   Colossus Mark 1   

  • first prototype shown to be working in December 1943 and was used by early 1944
  • mostly operated by women since men are deployed in the battlegrounds
  • made by British

   ENIAC   

  • Electronic Numerical Integrator and Computer
  • first programmable, general-purpose digital computer in 1946

   John Von Neumann Machine   

  • the architecture of today’s computer
  • the computers we use are based on this machine

   Smartphone   

  • also a computer wherein the architecture is based on Von Neumann

   Fortran   

  • FORmula TRANslation
  • first successful programming language
  • originally published in 1957
  • designed by John Backus in 1965
  • John W. Backus created the fortran PL in 1953 where he used IBM

  Microcomputers   first appeared on the market in 1975 (Commodore, Tandy, and Apple; much later,
IBM entered the market)

   Software   

  • interface between computer systems and the humans who use them
  • consists of programming instructions and data that tell the computer how to execute various tasks

First piece of software was written by computer scientist   Tom Kilburn   on June 21, 1948. Kilburn and
his colleague   Freddie Williams   had built one of the earliest computers, the   Manchester Small-Scale  
Experimental Machine   (aka the ”Baby”). The SSEM was programmed to perform mathematical
calculations using machine code instructions.

In 1958, statistician   John Tukey   coined the word   “software”  in an article about computer programming.
Other pioneering programming languages like Fortran, Cobol, BASIC, Pascal and C arrived over the next
two decades
